Estructura de Datos   Estructura de Datos 

1.-Tipos abstractos de datos (TAD): Introducción a los TAD y su relación con las estructuras de datos

2.-Estructuras de datos lineales: Estudio y aplicación de estructuras lineales como listas, pilas y colas

3.-Estructuras de datos no lineales: Estudio y aplicación de estructuras no lineales como árboles y grafos.

4.-Algoritmos de ordenamiento: Estudio y aplicación de algoritmos de ordenamiento como burbuja,
selección, inserción, merge sort, quicksort y heapsort

5.-Tablas hash: Estudio de las tablas hash como estructuras de datos eficientes para la búsqueda y recuperación de elementos

6.-Algoritmos de búsqueda: Estudio y aplicación de algoritmos de búsqueda como búsqueda secuencial, búsqueda binaria y búsqueda en árboles binarios.

7.-Análisis de complejidad: Estudio del análisis de complejidad temporal y espacial de los algoritmos y estructuras de datos

8.-Estructuras de datos avanzadas:estructuras de datos persistentes, estructuras de datos probabilísticas, estructuras de datos para grafos
