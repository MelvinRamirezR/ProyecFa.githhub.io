 Programacion Web   Programacion Web 

1.-HTML y CSS: Introducción a HTML (HyperText Markup Language) y CSS (Cascading Style Sheets)
para la estructura y el diseño de páginas web

2.-JavaScript: Estudio de JavaScript, un lenguaje de programación utilizado para agregar interactividad
y dinamismo

3.-Frameworks front-end: Introducción a frameworks front-end populares como React, Angular o Vue.js

4.-Diseño responsive: Estudio de técnicas y prácticas para crear diseños web que se adapten a diferentes dispositivos y tamaños de pantalla

5.-Accesibilidad web: Importancia y principios de la accesibilidad web, accesibles para personas con discapacidades visuales, auditivas o motoras.

6.-Backend y bases de datos: Introducción al desarrollo del lado del servidor utilizando lenguajes como PHP, Python o Node.js

7.-PI RESTful: Estudio de la arquitectura de servicios web RESTful y cómo crear y consumir APIs

8.-Seguridad web: Introducción a los conceptos básicos de seguridad web, incluyendo la protección contra ataques como Cross-Site Scripting (XSS), Cross-Site Request Forgery (CSRF) y SQL Injection
